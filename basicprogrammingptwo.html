Programming Basics

How Computers Work

ကွန်ပျူတာသည် အချက်အလက်များကို အားဖြင့် ကြီးကြီးမားမား အလုပ်လုပ်စေဖို့ သုံးသော စက်ဖြစ်သည်။ ထိန်းချုပ်မှုကို Programming ဖြင့် ပြုလုပ်သည်။

  • Input: ကွန်ပျူတာသို့ အချက်အလက်များကို ဝင်ရောက်စေခြင်း (e.g., ကီးဘုတ်, မောက်စ်)
  • Processing: CPU သည် အချက်အလက်များကို ဆောင်ရွက်ခြင်း
  • Output: အချက်အလက်များကို ပြန်လည် ထုတ်ပြခြင်း (e.g., မော်နီတာ, ပရင်တာ)
  • Storage: အချက်အလက်များကို သိမ်းဆည်းခြင်း (e.g., Hard Drive, SSD)

What is Programming?

Programming သည် ကွန်ပျူတာကို မိမိအလိုအလျောက် အလုပ်လုပ်စေဖို့ ရေးသားသော အစီအစဉ်များဖြစ်သည်။ Programmer များက ကွန်ပျူတာ ဘာလုပ်စေချင်သလဲ ဆိုတာကို Programming Language တွေကို အသုံးပြုပြီး ရေးသားပေးရသည်။

  • Programming သည် အစီအစဉ်များကို ရေးသားခြင်းဖြစ်သည်။
  • စက်များနှင့် ကွန်ပျူတာများကို ထိန်းချုပ်ရန် ရည်ရွယ်သည်။
  • Programming Language များကို အသုံးပြုပြီး ရေးသားရသည်။

Popular Programming Languages

Python

Python သည် လွယ်ကူပြီး အသုံးပြုရ အဆင်ပြေသော Programming Language တစ်ခုဖြစ်သည်။

Example:

name = "Zhorik"
age = 30
print("Name:", name)
print("Age:", age)

အသုံးများသော Application များ - YouTube, Instagram, Spotify

JavaScript

JavaScript သည် Web Development တွင် အဓိက အသုံးပြုသော Programming Language ဖြစ်သည်။

Example:

let name = "Zhorik";
let age = 30;
console.log("Name:", name);
console.log("Age:", age);

အသုံးများသော Application များ - Facebook, Twitter, LinkedIn

C++

C++ သည် အဆင့်မြင့်စွာ အစီအစဉ်ရေးရန် အသုံးပြုသည်။

Example:

#include <iostream>
using namespace std;

int main() {
    string name = "Zhorik";
    int age = 30;
    cout << "Name: " << name << endl;
    cout << "Age: " << age << endl;
    return 0;
}

အသုံးများသော Application များ - Adobe Products, Microsoft Office, Games (e.g., Counter-Strike)

How Does Programming Work?

Programming သည် အကြီးမားသော Project များကို အသေးစိပ် အစီအစဉ်များအဖြစ် ခွဲခြားပြီး အတူတကွ အလုပ်လုပ်သည်။ Programmer များက အစီအစဉ်များကို ရေးသားပြီး၊ Compiler (သို့) Interpreter များက အစီအစဉ်များကို ကွန်ပျူတာနားလည်သည့် Machine Code အဖြစ် ပြောင်းလဲပေးသည်။

  • Programmer များက အစီအစဉ်များကို ရေးသားသည်။
  • Compiler သို့မဟုတ် Interpreter များက Machine Code အဖြစ် ပြောင်းလဲသည်။
  • Machine Code သည် ကွန်ပျူတာ၏ နားလည်နိုင်သော ဘာသာစကားဖြစ်သည်။

Interesting Facts

  • Programming Languages များသည် ၁၉၅၀ ပြည့်နှစ်များမှ စတင်၍ ဖန်တီးခဲ့သည်။
  • Python သည် ၁၉၉၁ ခုနှစ်တွင် ဖန်တီးခဲ့ပြီး ကျောင်းသားများနှင့် စတင်လာသူများအတွက် အလွန်လွယ်ကူသော Language တစ်ခုဖြစ်သည်။
  • JavaScript သည် အင်တာနက်သုံးစွဲမှုအတွက် အဓိက အသုံးပြုသော Language ဖြစ်သည်။
  • HTML သည် မည်သည့် Programming Language မဆို မဟုတ်ဘဲ၊ စာမျက်နှာများ ဖော်ပြရန် အသုံးပြုသော Markup Language ဖြစ်သည်။
  • Programming သည် ကွန်ပျူတာ၏ နည်းပညာဖွံ့ဖြိုးမှုကို အခြေခံစေသည်။
  • Grace Hopper သည် စတင်ထိပ်သီး Compiler ကို ဖန်တီးသူဖြစ်သည်။
  • SQL သည် အချက်အလက်များကို စီစဉ်ရန် အသုံးပြုသော Database Language ဖြစ်သည်။
  • COBOL သည် ၁၉၅၉ ခုနှစ်တွင် ဖန်တီးခဲ့ပြီး မကြာသေးမီအထိ အသုံးပြုနေဆဲ ဖြစ်သည်။
